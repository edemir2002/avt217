House Instructions

  1. Launch Adobe XD
  2. Create a new document 1080x1000
  3. Using the Rectangle tool, create a blue rectangle in the center of the screen
  4. Using the Triangle tool, create a roof slightly larger than the rectangle,use a light blue
  5. Using the Rectangle tool, create a white door for the house and place it on the middle bottom edge of the blue rectangle
  6. Using the Triangle tool, create a front yard in light green. You can arrange the house in front to hide the top of the triangle
  7. Congratulations on the new house!
